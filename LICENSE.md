# Licencias

Este repositorio usa doble licencia para separar código y contenido educativo.

## Código

Todo el código fuente en `src/`, `examples/`, `tests/` y `benches/` se publica
bajo:

```text
MIT OR Apache-2.0
```

Ver `LICENSE-MIT` y `LICENSE-APACHE`.

## Contenido educativo

Los capítulos, diagramas, ejercicios, imágenes y documentación educativa se
publican bajo:

```text
Creative Commons Attribution-ShareAlike 4.0 International
CC BY-SA 4.0
```

Ver `LICENSE-CC-BY-SA-4.0.md`.

## Intención

El código debe ser reutilizable en proyectos reales. El contenido debe seguir
siendo abierto, atribuible y compartido bajo la misma licencia.
